# Html to PDF

Easy conversion of web pages to pdf and saves it to a pdf file.


# Requirements and install

*	pdfkit
*	wkhtmltopdf


## Install packages.
```
$ pip install -r requirements
```

## Install 'wkhtmltopdf'.
```
$ sudo dnf install wkhtmltopdf
```



# Run
```
$ python html_to_pdf.py
```
